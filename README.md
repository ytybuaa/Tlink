## Tlink
TLink 是一个基于 RAG（检索增强生成）的联想知识库。不同于传统的关键词搜索，TLink 利用 LLM 进行意图识别与结果精排，支持模糊搜索与语义搜索。项目核心目标是：避免重复造轮子，快速查询历史经验，追踪个人想法的演进脉络。
